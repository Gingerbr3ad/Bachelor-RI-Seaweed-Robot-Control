# Bachelor RI Seaweed Robot Control
Robot control software for controlling the KUKA KR 10 R900-2 robot via an external PC running ROS. This project was developed for a bachelor exercise as a part of the Automation and Intelligent Systems bachelor's degree programme at NTNU Trondheim.

The raspberry pi is connected to the blkspt wifi and its ip address is set to 10.0.0.31
